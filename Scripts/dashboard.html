💰 Сделки (Deals)

События с суммой цен ниже порога — арб уже сложился, бот будет файрить (или симулировать в DRY_RUN). Порог динамический: Polymarket 94.8¢–99.3¢ по формуле 1 − (taker_fee + 0.003 slippage), clamped в этот диапазон (floor 94.8 — для high-fee маркетов ≥4.9%, cap 99.3 — для нулевой комиссии; politics ≈ 97¢, sports ≈ 95-97¢, fee=0 ≈ 99¢); Limitless 99¢ — feeRateBps=300 по rank (Bronze 3%), но effectiveFeeBps=0% текущая промо для нового аккаунта (verified live, server возвращает effective в response); SX Bet 98¢ (2% на winnings, type=1/type=52 markets). Реальный профит = (1 − sum) × balance − fees − slippage.

🎯 Почти-арбитражи (NEAR pool)

События где сумма цен близка к порогу, но ещё не пересекла. Порог динамический по платформе: Polymarket 94.8¢–99.3¢ по 1 − (taker_fee + 0.003) (floor 94.8¢ для маркетов с fee ≥4.9% — обычно sports; cap 99.3¢ для fee=0 промо), Limitless/Kalshi/SX — фикс. Структуры A/B показываются в окне до +7¢ от порога; структура C — только до +3¢ (она per-market и без cap'а захламляла бы pool). Polymarket — обновления через WebSocket в реальном времени; Kalshi/SX — REST polling. Когда sum просядет ниже порога — событие переедет в Deals.
